## Time Complexity

| Operation             | Big-O     | Note |
| --------------------- | --------- | ---- |
| Access                | O(1)      |      |
| Search                | O(n)      |      |
| Search (sorted array) | O(log(n)) |      |
| Insert                | O(n)      |      |
| Insert (at the end)   | O(1)      |      |
| Remove                | O(n)      |      |
| Remove (at the end)   | O(1)      |      |
